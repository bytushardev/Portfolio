# 💼 Tushar's Portfolio

Welcome to my personal portfolio website built with **React** and **Vite**.

## 🚀 About

This portfolio showcases my skills, projects, and journey as a frontend developer. It is designed with a clean, modern, and responsive user interface.

## ✨ Features

- Responsive Design
- Modern UI
- About Me Section
- Projects Showcase
- Contact Page

## 🛠 Tech Stack

- React
- JavaScript
- HTML5
- CSS3
- Vite

## 📂 Installation

```bash
git clone https://github.com/bytushardev/Portfolio.git

cd Portfolio

npm install

npm run dev
```
